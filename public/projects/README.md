# Project Images

Add your project screenshots/images here.

## Image Guidelines

### Recommended Specifications
- **Format:** PNG, JPG, or WebP
- **Dimensions:** 1200x630px (16:9 aspect ratio works best)
- **File size:** Under 500KB for optimal performance
- **Quality:** High resolution but optimized for web

### Naming Convention
Use descriptive names matching your project:
- `rantale.png` - Rantale Web Novel Platform
- `cryptotip.png` - CryptoTip Crypto Donations

### How to Add Images

1. **Take a screenshot** of your project
2. **Optimize the image** using tools like:
   - [TinyPNG](https://tinypng.com/)
   - [Squoosh](https://squoosh.app/)
   - [ImageOptim](https://imageoptim.com/)
3. **Save the file** in this directory (`/public/projects/`)
4. **Update** `/data/projects.ts` with the image path:
   ```typescript
   image: "/projects/your-image.png"
   ```

### Placeholder Images

If you don't have images yet, you can:
1. **Use screenshots** from your live projects
2. **Create mockups** using Figma or Canva
3. **Use placeholder services** like:
   - https://placehold.co/1200x630/png
   - https://via.placeholder.com/1200x630

### Example

```typescript
{
  title: "My Awesome Project",
  image: "/projects/my-awesome-project.png",
  // ... other properties
}
```

## Tips

- Use consistent dimensions for all project images
- Capture the most representative screen of your project
- Ensure text in screenshots is readable
- Consider using your project's landing page or dashboard
- Update images when you redesign your projects
